# Sensoriamento Remoto - Conceitos

Os dados do monitoramento ambiental são coletados atraves de sensores instalados em ambientes aquaticos, terrestres, aereos e orbitais. Os sensores sao usados para identificar, mapear e monitorar condições ambientais. Os estudos de monitoramento servem para avaliar a mudança histórica de uma vegetação, o controle de pragas na agricultura, a população da vegetação aquática, entre outros.

Os estudos de sensoriamento remoto é interdisciplinar e apresenta diferentes vertentes. Os sensotes captam a radiação eletromagnética do solo, da água, da vegetação, etc.
